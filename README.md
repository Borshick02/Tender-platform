
##Тендеры Pro
Веб-приложение для автоматического поиска, сбора и анализа тендеров с российских и зарубежных торговых площадок.

##Что умеет
Параллельный парсинг 18 площадок: RTS-Tender, RUTEND, Synapse, Tender.Pro, РосТендер, ТЭК-Торг, B2B-Center, Сбербанк-АСТ, ЗаказРФ, Росэлторг, Фабрикант, ЭТП ГПБ, Goszakup.kz и зарубежные B2B-площадки (Made-in-China, Global Sources, HKTDC, DHgate и др.)
Фильтрация результатов по датам и минимальному остатку дней до дедлайна
ML-аналитика по каждому тендеру: прогноз итоговой цены (линейная регрессия), оценка риска и репутация заказчика
AI-анализ рисков через языковую модель (OpenRouter / Gemma 2 9B) с fallback на локальные эвристики
Личный кабинет с историей заявок и избранным
Real-time статус выполнения поиска
